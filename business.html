API 특징

왜 무비서 API인가?

검증된 AI 추천 엔진을 쉽고 빠르게 연동하세요

빠른 응답 속도

평균 300ms 이내 응답. 사용자 경험을 해치지 않는 실시간 추천을 제공합니다.

하이브리드 AI

SBERT 콘텐츠 분석 + LightGCN 협업 필터링으로 정확한 개인화 추천을 제공합니다.

시간 맞춤 추천

사용자의 가용 시간에 맞는 영화 조합을 자동으로 계산하여 추천합니다.

10,000+ 영화 DB

TMDB 기반 풍부한 메타데이터와 한국어 정보를 포함한 영화 데이터베이스.

안정적인 서비스

99.9% 가동률 보장. 클라우드 기반 인프라로 안정적인 서비스를 제공합니다.

간편한 연동

RESTful API로 어떤 플랫폼에서도 쉽게 연동. 상세한 문서와 예제 코드 제공.

API 문서

간단한 연동, 강력한 추천

단 한 번의 API 호출로 AI 기반 영화 추천을 받아보세요

영화 추천 API

사용자가 좋아하는 영화 ID와 가용 시간을 전달하면, AI가 분석하여 최적의 영화 조합을 추천합니다.

  • Track A: 선호 장르 맞춤 추천
  • Track B: 장르 확장 다양성 추천
  • 런타임 합계 자동 계산
  • 포스터, 줄거리 등 메타데이터 포함
  • 성인물 필터링 옵션
POST https://api.moviesir.cloud/b2b/recommend Content-Type: application/json X-API-Key: your_api_key { "liked_movies": [550, 27205, 157336], "runtime_limit": 360, "genres": ["액션"], "exclude_adult": true }
